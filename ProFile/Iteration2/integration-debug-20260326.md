## ❌ 任务管理API验证

### GET /api/tasks

**请求**:
```bash
curl -H "Authorization: Bearer {token}" http://localhost:3001/api/tasks
```

**响应**:
```json
{
  "code": 500,
  "msg": "column \"deleted_at\" does not exist",
  "data": null,
  "trace_id": "default",
  "timestamp": "2026-03-26T07:18:50.249Z"
}
```

**问题**: 数据库表结构与Sequelize模型不一致

**根本原因**: tasks表的Sequelize模型定义缺少deleted_at字段
