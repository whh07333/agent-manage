# OpenClaw AI项目管理系统 - PC端高保真设计说明
**版本**: v1.0  
**设计人**: Tina  
**创建日期**: 2026-03-12  

---

## 概述
OpenClaw AI项目管理系统是一个面向AI Agent优先的项目管理平台，同时为人类用户提供可视化监控界面。设计遵循企业级专业风格，注重数据的清晰展示和操作的高效性。

---

## 整体布局架构
```
┌─────────────────────────────────────────────────────────┐
│ 顶部导航栏 (高度: 64px) - Logo/用户/通知/设置          │
├───────────────────────┬─────────────────────────────────┤
│ 侧边栏 (宽度: 240px)  │ 主内容区域 (自适应宽度)         │
│ - 仪表盘             │ - 页面标题/面包屑导航            │
│ - 项目列表           │ - 操作工具栏 (筛选/搜索/新增)    │
│ - 任务管理           │ - 主要内容区域 (卡片/表格/图表)  │
│ - 数据统计           │ - 底部信息/分页控件              │
│ - 审计日志           │ - 辅助信息/状态指示              │
│ - 系统设置           │                                   │
└───────────────────────┴─────────────────────────────────┘
```

## 页面设计详情

### 1. 仪表盘 (Dashboard)
**路径**: `/`  
**优先级**: P0  
**用途**: 项目整体概览，快速了解项目状态和关键指标

#### 核心设计元素
- **页面布局**: 24列栅格系统
- **头部区域**: 页面标题、统计时间范围选择、刷新按钮
- **关键指标区**: 4个统计卡片
- **图表区**: 2个核心图表 + 1个数据看板
- **任务列表区**: 待处理任务列表

#### 设计要点
1. **统计卡片** (4个卡片布局)
   - **活跃项目数**: 显示当前活跃项目数量，点击可跳转项目列表
   - **待验收任务**: 显示等待验收的任务数量，高亮显示
   - **阻塞任务数**: 显示阻塞状态的任务数量，红色警告
   - **平均交付周期**: 显示项目平均交付周期，历史趋势比较

2. **项目进度图**
   - **图表类型**: 燃尽图
   - **数据范围**: 最近1个月的任务完成趋势
   - **交互**: 悬停显示详细数据，点击可跳转统计页面
   - **样式**: 蓝色渐变，辅助网格线

3. **Agent工作负载**
   - **图表类型**: 条形图
   - **数据范围**: 所有活跃Agent的任务数量分布
   - **交互**: 悬停显示具体任务数，点击可跳转Agent管理
   - **样式**: 多色系条形，区分不同Agent类型

4. **风险看板**
   - **布局**: 3列布局，分别显示待分配、进行中、阻塞任务
   - **样式**: 卡片式布局，状态颜色编码
   - **交互**: 点击卡片可跳转任务详情页
   - **更新**: 实时更新任务数量

### 2. 项目列表页 (Project List)
**路径**: `/projects`  
**优先级**: P0  
**用途**: 查看所有项目，筛选、搜索、创建新项目

#### 核心设计元素
- **页面布局**: 24列栅格系统
- **操作工具栏**: 搜索框、筛选按钮、创建项目按钮
- **项目列表区**: 卡片式布局，支持网格/列表视图切换
- **分页控件**: 底部分页，每页显示12个项目
- **筛选面板**: 侧边栏筛选，支持多种条件组合

#### 设计要点
1. **项目卡片** (卡片布局)
   - **项目信息**: 项目名称、负责人、优先级、状态
   - **进度条**: 项目完成进度，颜色编码（蓝色/橙色/红色）
   - **任务统计**: 总任务数/待分配/进行中/完成
   - **操作按钮**: 详情/编辑/归档/删除

2. **项目状态** (颜色编码)
   - **进行中**: 蓝色进度条，项目活跃标识
   - **逾期**: 红色进度条，警告图标
   - **已完成**: 绿色进度条，完成标识
   - **已归档**: 灰色卡片，降低对比度

3. **操作工具栏**
   - **搜索**: 支持项目名称、负责人、标签搜索
   - **筛选**: 支持状态、优先级、创建时间筛选
   - **视图切换**: 网格/列表视图切换
   - **批量操作**: 支持批量归档、删除

   - **时间范围**: 操作发生时间范围
   - **操作类型**: 按操作类型筛选
   - **操作人**: 按操作Agent筛选
   - **操作对象**: 按操作对象筛选
