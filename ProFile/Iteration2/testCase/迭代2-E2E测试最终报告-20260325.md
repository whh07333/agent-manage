### 建议的改进方法

1. **使用data-testid属性**:
   ```tsx
   <button data-testid="create-project-button">创建项目</button>
   <button data-testid="archive-project-button">归档</button>
   <button data-testid="delete-project-button">删除</button>
   ```

2. **使用Ant Design的CSS类名**:
   ```typescript
   const createButton = page.locator('.ant-btn-primary');
   const archiveButton = page.locator('button:has-text("归档")');
   ```

3. **增加等待时间**:
   ```typescript
   await page.waitForLoadState('networkidle');
   await page.waitForTimeout(2000);
   ```
